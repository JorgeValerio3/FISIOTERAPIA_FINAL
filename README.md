# 🌊 UFAAL - Plataforma Oficial

**Unión de Fisioterapia Acuática de América Latina**

Este proyecto es la presencia digital oficial de **UFAAL**, diseñada para centralizar la formación, investigación y colaboración de profesionales de la fisioterapia acuática en toda la región. El enfoque principal es ofrecer una experiencia de usuario sobria, académica y de alto rendimiento.

## ✨ Características Principales

* **Mapa Interactivo de Países Miembros:** Visualización dinámica de la red latinoamericana.
* **Gestión de Formación e Investigación:** Repositorio de recursos académicos y eventos.
* **Sección de Noticias Dinámica:** Vistas detalladas para las últimas actividades de la unión.
* **Diseño "Apple-Style":** Interfaz minimalista, profesional y altamente responsiva (paleta de azules y blancos).

## 🛠️ Stack Tecnológico

* **Frontend:** React con TypeScript.
* **Backend:** Node.js + Express.js.
* **Funcionalidades Backend:** * Servicio de mensajería (Mailer) para contacto directo.
* Gestión de archivos descargables (documentación académica).


* **Estilos:** CSS Moderno / Tailwind (enfocado en estética limpia).

## 🚀 Instalación y Uso

1. **Clonar el repositorio:**
```bash
git clone https://github.com/tu-usuario/ufaal-project.git

```


2. **Configurar el Frontend:**
```bash
cd client
npm install
npm start

```


3. **Configurar el Backend:**
```bash
cd server
npm install
npm run dev

```



## 📖 Arquitectura del Proyecto

El proyecto sigue una estructura modular para facilitar la escalabilidad:

* `/src/components`: Componentes reutilizables con tipado estricto.
* `/src/pages`: Secciones principales (Quiénes Somos, Actividades, Investigación).
* `/api`: Endpoints para la gestión de correos y descargas.
